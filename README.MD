# React Marvel App

React Marvel App is a React App created for the React Starter Talk presented at November tech by SENAC.


## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install React Marvel App.

```bash
yarn
```

## To Run

```bash
yarn dev
```
